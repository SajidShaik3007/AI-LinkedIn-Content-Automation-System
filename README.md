
<h1>🚀 AI LinkedIn Content Automation System</h1>

<p>
This project is a fully automated AI-powered system that generates high-quality LinkedIn posts
and publishes them automatically along with AI-generated images — all on a scheduled basis.
</p>

<hr>

<h2>🎯 Real-World Use Case</h2>

<div class="highlight">
Consistently posting high-quality content on LinkedIn is time-consuming.
This system solves that by automatically generating engaging posts and visuals,
allowing individuals and businesses to maintain an active presence without manual effort.
</div>

<hr>

<h2>🤖 Core Features</h2>

<ul>
  <li>⏰ Scheduled post automation</li>
  <li>🧠 AI-generated LinkedIn content</li>
  <li>✍️ Human-like writing with strong hooks</li>
  <li>🖼️ AI-generated images using Hugging Face API</li>
  <li>📊 Content management via Google Sheets</li>
  <li>🚀 Automatic posting to LinkedIn</li>
</ul>

<hr>

<h2>🧠 Workflow Overview</h2>

<p>
1. Workflow runs on a scheduled trigger (daily at a set time)  
2. Fetches content data from Google Sheets:
</p>

<ul>
  <li>Topic</li>
  <li>Description</li>
  <li>Image Prompt</li>
</ul>

<p>
3. AI generates a LinkedIn post:
</p>

<ul>
  <li>Uses Topic as title</li>
  <li>Creates a strong hook</li>
  <li>Generates engaging content (150+ words)</li>
</ul>

<p>
4. Image is generated using Hugging Face API  
5. Combines text + image  
6. Automatically publishes post to LinkedIn  
</p>

<hr>

<h2>⚙️ Tech Stack</h2>
